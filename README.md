# 股票预警系统

> Electron+Vue+Ant Design 股票预警系统客户端



## 技术栈
  * Electron
  * vue
  * vuex
  * vue-router
  * echarts
  * axios
  * ant-design-vue
  * dayjs



## 运行
  &emsp;&emsp;安装依赖
npm install<br />
  &emsp;&emsp;运行程序
npm run start
## 开发说明
* render>views>dataV	
主要页面
* render>utils
公共函数方法、功能
* render>store
vuex全局变量
* render>store>APP.vue
项目启动根页面
* render>plugins>andt-design-vue.js
项目主要使用组件
* render>router>index.js
项目路由地址
* package.json
项目使用的所有插件及版本
* 项目同时可以使用npm及yarn运行
